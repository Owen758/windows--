# 实施计划 — 历史粘贴板 v1.0

## 开发原则

1. 每阶段只做 1-2 个模块，完成后验证再推进
2. 每个阶段结束更新 devlog
3. 文档如有变更，及时同步

---

## 阶段 0：项目脚手架（预计 1-2 小时）

**目标：** 搭好项目骨架，能看到窗口运行

**任务：**
- [ ] `npm init` + 安装 electron 依赖
- [ ] 创建 `src/main/index.js` — 基础主进程（创建窗口）
- [ ] 创建 `src/preload.js` — preload 脚本
- [ ] 创建 `src/renderer/index.html` — 空白页面，显示 "Hello 历史粘贴板"
- [ ] 创建 `src/renderer/style.css` — 基础样式
- [ ] 配置 `package.json` 启动脚本
- [ ] 验证：`npm start` 能打开窗口

**产出：** 可运行的空 Electron 窗口

---

## 阶段 1：数据库 + 剪贴板监控（预计 2-3 小时）

**目标：** 后台能记录复制内容到 SQLite

**任务：**
- [ ] 安装 `better-sqlite3` 依赖
- [ ] 创建 `src/main/database.js` — 建表、CRUD 操作
- [ ] 创建 `src/main/clipboard.js` — 剪贴板轮询监控
- [ ] 实现文字复制记录
- [ ] 实现图片复制记录（保存图片文件）
- [ ] 实现内容去重
- [ ] 验证：复制文字和图片，确认数据库有记录

**产出：** 剪贴板后台监控功能

---

## 阶段 2：主窗口 UI + 卡片列表（预计 3-4 小时）

**目标：** 主窗口展示所有历史记录卡片

**任务：**
- [ ] 实现 IPC 通信（main ↔ renderer）
- [ ] 创建 `src/renderer/app.js` — 获取历史数据并渲染
- [ ] 创建 `src/renderer/components/card.js` — 文字/图片卡片组件
- [ ] 实现卡片列表（时间倒序）
- [ ] 实现滚动加载（分页，每次加载 50 条）
- [ ] 卡片 hover 效果
- [ ] 点击卡片复制到剪贴板
- [ ] 验证：复制多条内容后打开窗口，卡片正确展示

**产出：** 可展示历史记录的主窗口

---

## 阶段 3：搜索 + 置顶 + 删除（预计 2-3 小时）

**目标：** 核心交互功能完整

**任务：**
- [ ] 创建 `src/renderer/components/search.js` — 搜索框，实时过滤
- [ ] 实现置顶/取消置顶（数据库更新 + 列表刷新）
- [ ] 实现删除（单条删除）
- [ ] 置顶卡片分组显示在列表顶部
- [ ] 操作按钮 UI（卡片 hover 显示）
- [ ] 验证：搜索、置顶、删除均正常

**产出：** 完整的卡片交互

---

## 阶段 4：自动清理（预计 1-2 小时）

**目标：** 过期内容自动清理

**任务：**
- [ ] 创建 `src/main/cleaner.js` — 清理逻辑
- [ ] 启动时检查 + 定时检查（每 12 小时）
- [ ] 清理时同步删除图片文件
- [ ] 验证：修改系统时间测试清理逻辑

**产出：** 自动清理过期记录

---

## 阶段 5：系统托盘 + 快捷键 + 开机自启（预计 2-3 小时）

**目标：** 软件能常驻后台，方便唤起

**任务：**
- [ ] 创建托盘图标和右键菜单
- [ ] 注册全局快捷键 Ctrl+Shift+V
- [ ] 窗口显示/隐藏逻辑（关闭到托盘，不退出）
- [ ] 创建 `src/main/autostart.js` — 开机自启
- [ ] 应用图标 .ico 设计（简单版）
- [ ] 验证：托盘和快捷键交互正常

**产出：** 完整的后台常驻 + 唤起体验

---

## 阶段 6：设置面板 + 主题自定义（预计 3-4 小时）

**目标：** 用户可自定义软件外观和行为

**任务：**
- [ ] 创建设置面板 UI（子窗口）
- [ ] 设置存取（settings.json 读写）
- [ ] 保留天数设置
- [ ] 快捷键自定义录入
- [ ] 开机自启开关
- [ ] 浅色/深色主题切换
- [ ] 主色调自定义（预设 + 取色器）
- [ ] 创建 `src/renderer/theme.js` — 主题管理（CSS 变量）
- [ ] 验证：各项设置生效并持久化

**产出：** 完整的设置功能

---

## 阶段 7：打包测试（预计 2-3 小时）

**目标：** 生成可安装的 .exe 文件

**任务：**
- [ ] 配置 electron-builder（生成 Windows .exe）
- [ ] 安装包测试（Win10 / Win11）
- [ ] 高 DPI 测试
- [ ] 边缘情况测试（空数据库、大量记录等）
- [ ] 操作指南（README 或 内置帮助）
- [ ] GitHub 仓库创建和代码推送

**产出：** 可交付的安装包
