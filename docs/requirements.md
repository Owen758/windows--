# 需求文档 — 历史粘贴板 v1.0

## 一、项目定位

一款运行在 Windows 10/11 上的剪贴板历史管理工具。静默运行在后台，自动记录用户复制的内容，需要时通过托盘或快捷键呼出，快速查找和复用历史记录。

## 二、目标用户

日常办公、学习、写作等需要频繁复制粘贴的 Windows 用户。不需要技术背景，开箱即用。

## 三、功能需求

### F1 — 剪贴板监控

- 系统启动后自动在后台运行
- 实时监控剪贴板变化，记录新复制的内容
- 支持内容类型：
  - 文字（纯文本、富文本）
  - 图片（PNG、JPG、BMP、GIF）
- 去重：连续复制相同内容不重复记录
- 不记录某些应用的选项（v2 考虑）

### F2 — 历史记录展示

- 以卡片列表形式展示
- 按时间倒序排列（最新的在最上面）
- 每张卡片显示：
  - 内容预览（文字截取前若干字符，图片显示缩略图）
  - 复制时间（相对时间 + 绝对时间）
  - 内容类型图标
- 点击卡片：复制内容到剪贴板，可直接粘贴使用
- 图片卡片：点击可放大预览

### F3 — 保留天数

- 设置选项中可选择：1天 / 3天 / 5天
- 过期的非置顶内容自动清理
- 每次启动时和每 12 小时检查一次

### F4 — 置顶

- 卡片右键菜单或按钮提供"置顶/取消置顶"
- 置顶卡片在列表顶部区域固定显示
- 标记视觉区分（如边框高亮或图钉图标）
- 置顶内容不受保留天数影响，不被自动清理

### F5 — 删除

- 卡片右键菜单或按钮提供"删除"
- 支持单张删除
- 可选"清空全部"（需二次确认）

### F6 — 搜索

- 顶部搜索框，实时过滤
- 对文字内容进行关键词匹配（模糊搜索）
- 搜索结果仍按时间倒序
- 置顶卡片在所有搜索结果之上

### F7 — 主题自定义

- 预设若干主题色方案（浅色/深色各几套）
- 用户可自定义主色调
- 主题设置实时预览

### F8 — 系统托盘

- 托盘图标常驻 Windows 任务栏通知区域
- 左键单击：显示/隐藏主窗口
- 右键菜单：打开、设置、退出
- 应用关闭到托盘（不直接退出）

### F9 — 全局快捷键

- 默认快捷键：Ctrl+Shift+V
- 在设置中可自定义快捷键
- 快捷键行为：显示/隐藏主窗口

### F10 — 开机自启

- 默认开启
- 在设置中可开关
- 使用 Windows 注册表或启动文件夹实现

## 四、非功能需求

### N1 — 性能
- 软件空闲内存占用 < 150MB
- 历史记录总数 < 5000 条时保持流畅
- 启动时间 < 3 秒

### N2 — 数据安全
- 所有数据存储在本地
- 不上传任何内容到网络
- v1 不做加密，后续版本考虑

### N3 — 兼容性
- Windows 10 版本 1809 及以上
- Windows 11 全部版本
- 高 DPI 屏幕正常显示

### N4 — 安装与更新
- 提供 .exe 安装包
- 安装路径用户可选
- v2 考虑自动更新
